易有太极，是生两仪，两仪生四象，四象生八卦，八卦定吉凶，吉凶生大业

本app共有“姓名”“八字”“风水”“号码”四个功能页，每页功能如下:

![Image](https://raw.githubusercontent.com/jinanwanjing/yijingtool/main/logo.png)

### 功能

1. “姓名”页可以进行姓名分析和起名建议
2. “八字”页可以进行生辰数字的转换
3. “风水”页可以进行房屋家具方向的计算
4. “号码”页可以进行手机、车牌、车位等号码的分析
